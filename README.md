## 📌 What this is

This app helps you explore multi-agent systems with LangChain on Windows. It shows how separate AI agents can work on one task, share work, and return a result you can use.

Use this if you want to:
- try an AI app on your PC
- learn how agent orchestration works
- test examples with OpenAI or Anthropic models
- follow a guided tutorial without building from scratch

## 🖥️ What you need

Before you start, make sure you have:
- a Windows 10 or Windows 11 computer
- an internet connection
- enough free space for the app and model files
- a modern browser if the app opens in a local web view
- an API key for the AI service you want to use

The app is built for normal desktop use. You do not need to write code to run the release package.

## 💾 Install on Windows

If you download an `.exe` file:
1. Open the file from your Downloads folder
2. If Windows asks for permission, choose Yes
3. Follow the setup prompts
4. Wait for the install to finish
5. Open the app from the Start menu or desktop shortcut

If you download a `.zip` file:
1. Right-click the file
2. Choose Extract All
3. Pick a folder you can find again, such as Desktop or Documents
4. Open the extracted folder
5. Start the app by double-clicking the main `.exe` file inside

If Windows shows a security prompt, choose the option that lets you keep going only if you trust the release source.

## 🧠 What the app does

This project is a tutorial and code example set for multi-agent AI systems. It shows how to split work between agents so each one has a job.

You may see examples for:
- task planning
- agent handoff
- message passing
- shared memory
- model choice between OpenAI and Anthropic
- TypeScript-based app structure

The app is meant to help you understand how AI agents can work as a team. One agent can plan. Another can gather facts. Another can write the final answer.

## 🛠️ Troubleshooting

If the app does not start:
1. Check that you downloaded the latest release
2. Make sure the file finished downloading
3. Try running it again as an administrator
4. Reboot your PC and try once more

If the app opens but does not answer:
1. Check your API key
2. Make sure your internet connection works
3. Confirm that your account has access to the model you chose
4. Try switching between OpenAI and Anthropic if both are available

If you see an error about missing files:
1. Extract all files if you downloaded a zip
2. Keep the folder structure the same
3. Do not move single files out of the app folder
4. Run the main app file from the extracted folder

If the interface looks blank:
1. Close the app
2. Open it again
3. Wait a few seconds for the page to load
4. Check that your browser or Windows display settings are normal

- check which model provider is active before you run a task

## 🧪 Example workflow

A simple run may look like this:
1. You enter a goal, such as writing a short plan
2. The planner agent breaks the task into parts
3. A worker agent gathers or drafts content
4. Another agent reviews the result
5. The app shows the final output

This structure makes it easier to see how one task can move through several AI steps.

## 📘 If you want to learn more

Start with the sample tasks inside the app. Then change one thing at a time:
- switch the model
- change the prompt
- add a new agent step
- compare the output

That approach helps you see how each part affects the result without making the process hard to follow
